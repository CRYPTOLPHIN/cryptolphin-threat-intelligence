Cryptolphin Threat Intelligence

Open security intelligence for protecting Web3 users from scams, phishing and malicious blockchain activity.

Mission

Cryptocurrency transactions are irreversible. Prevention is the strongest form of security.

Cryptolphin provides privacy-first threat intelligence designed to help users, wallets, developers and security researchers identify malicious infrastructure before harm occurs.

Current Coverage

- Ethereum
- BNB Chain
- Polygon
- Arbitrum
- Optimism
- Avalanche
- Fantom

Threat Categories

Currently tracked:

- Scam wallet addresses
- Phishing domains
- Address poisoning attacks
- Impersonation campaigns
- Social engineering indicators

Verification Process

Threat entries are not added based on single reports.

Cryptolphin uses a verification process requiring confirmation from multiple independent sources before inclusion.
